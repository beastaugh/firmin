Firmin.js

Firmin is a JavaScript animation library that uses CSS transforms and transitions to create smooth, hardware-accelerated animations.

Supported platforms

Currently, the only supported platforms are those that use a recent version of the WebKit engine: Safari 4, Mobile Safari (i.e. iPhone and iPad) and Google Chrome. This is because CSS transitions are only available for WebKit browsers. Once this feature is more widely supported an effort will be made to make Firmin compatible with other browsers.

Reference

animate Firmin.animate(element, transform, [duration])

The transform object should be a description of the transformation to be carried out on the element. For example, this transform would move the element 400 pixels to the right and double its height. 

var el = document.getElementById("test"),
    tf = {
        translateX: "400px",
        scaleY:     2
    };

Firmin.animate(el, tf);

Instead of using animate with a transform argument, one can use the transform functions directly. Each transform function (e.g. translateX, skewY) accepts a transform value as its second argument. This value is also what should be used in the transform object when calling animate. In other words, 

Firmin.animate(el, {skewX: "1rad"});

is equivalent to

Firmin.skewX(el, "1rad");

The optional duration argument specifies the duration of the animation in seconds; the default value is 0.2.
